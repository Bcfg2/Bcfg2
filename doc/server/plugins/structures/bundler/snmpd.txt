snmpd
=====

A simple bundle for a SNMP daemon with a package, a service and a
configuration file.

.. code-block:: xml

    <Bundle>
            <Package name="snmpd"/>
            <Service name="snmpd"/>
            <Path name="/etc/snmp/snmpd.conf"/>
    </Bundle>
